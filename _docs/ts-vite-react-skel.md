## Generation Notes

- Uses `npm create vite@latest` to scaffold base project
- Removes generated `vite.config.ts` and `tsconfig.node.json`
- Copies skeleton versions with Vitest configuration
- Excludes `package.json`, `package-lock.json`, `tsconfig.json` from merge

## Generation

From repo root:
```bash
make gen-ts-vite-react NAME=<target-path>
```

From anywhere:
```bash
_bin/skel-gen ts-vite-react <target-path>
```

From skeleton dir:
```bash
./gen <target-path>
```


## Merge Script

This skeleton uses an executable `merge` script referenced by its Makefile as `MERGE := $(SKEL_DIR)/merge`. It copies auxiliary files into the generated project without overwriting generator-owned files.

### Merge Script Exclusions

- `package.json`
- `package-lock.json`
- `tsconfig.json`
- `tsconfig.node.json`
- `vite.config.ts`
